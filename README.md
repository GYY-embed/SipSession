# SipSession

SipSession 是一个面向 iPhone + Apple Watch 的酒局过程记录产品（MVP）文档仓库，聚焦「低成本记录 + 结束复盘」。

## 项目目标
- 在酒局进行中，通过 Apple Watch 快速记录饮酒与排尿事件，并查看心率趋势。
- 在 iPhone 端完成配置、状态控制与报告生成。
- 在酒局结束后输出可复盘的统计结果与恢复时间区间（非医疗结论）。

## MVP 核心功能
- 酒局生命周期：开始、暂停、继续、结束（含自动结束兜底）。
- 双端协同：iPhone/Watch 状态同步，iPhone 可接管当前 Session。
- Watch 记录：
  - `+1` 计杯
  - `🚻 +1` 排尿记录
  - 实时心率与趋势展示
- iPhone 配置：
  - 预置饮品模板管理（MVP 固定 5 种）
  - 开局前选择本次可用饮品
  - 局中“喝点什么”配置（仅影响后续记录）
  - 局中单杯微调（杯量/ABV/名称）
- 报告与分享：
  - 自动生成酒局报告（杯数、酒精 g、液体 ml、时长、心率摘要等）
  - 生成分享图片并保存到系统相册
  - 报告展示头像与名称

## MVP 非目标
- 不提供安全驾驶、可继续饮酒等安全/医疗判断。
- 不做自动排尿识别。
- 不做社交平台一键发布（仅保存到相册）。
- 不支持用户自定义饮品模板或图标。
